Mise en place du serveur web

SAÉ S1.03 – BUT Informatique

IUT de Paris – Rives de Seine

Pour héberger le site, Apache a été choisi car il s’agit d’un serveur web fiable, largement utilisé et open source. Nous l’avons donc installé sur notre machine virtuelle.

Installation d’Apache sur la machine virtuelle

Nous avons ensuite lancé Apache avec la commande suivante et configuré le service pour qu’il démarre automatiquement au démarrage de la VM.

Lancement du serveur Apache et activation du démarrage automatique

Pour stocker nos pages HTML, nous avons créé un dossier dédié.

Création du dossier pour les pages HTML

Nous avons donné la propriété du dossier à l’administrateur et changé les droits afin qu’il ait un accès complet pour pouvoir transférer les fichiers depuis PhpStorm vers la VM.

Propriété et droits du dossier configurés pour l’administrateur

Avec la commande ip a, nous avons récupéré l’adresse IP de la machine virtuelle.

Enfin, nous avons configuré PhpStorm avec l’adresse IP et le port (ici le port 22) pour envoyer directement les fichiers vers notre VM. Le serveur web est désormais opérationnel.

Configuration de PhpStorm pour le transfert des fichiers vers la VM
