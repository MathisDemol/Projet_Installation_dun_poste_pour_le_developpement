Présentation de la SAÉ

Cette SAÉ a pour objectif de nous initier aux bases de l’administration des systèmes informatiques à travers la mise en place d’un environnement de développement complet.

Le travail repose sur la création et la configuration d’une machine virtuelle sous Linux, l’installation d’un serveur web ainsi que le déploiement d’un site web respectant les standards du développement web moderne.

Objectifs du projet

Virtualisation

Comprendre le principe de la virtualisation et créer une machine virtuelle sous Linux.

Administration système

Gérer les utilisateurs, les groupes et les droits du système.

Développement web

Déployer un site web responsive avec HTML, CSS et Bootstrap.
